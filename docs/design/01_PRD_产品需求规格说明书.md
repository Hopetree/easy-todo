# 01_PRD_产品需求规格说明书

**项目名称**：Easy Todo（浏览器插件 TODO 管理器）

---

## 一、项目概述

Easy Todo 是一款浏览器插件形式的 TODO 任务管理工具，面向个人用户的日常任务管理需求。支持多列表管理、丰富任务属性、关键词搜索与字段筛选，并提供完整数据的导出/导入功能以实现数据持久化。

---

## 二、目标用户

- **主要用户**：个人用户，用于日常任务管理
- **使用场景**：工作待办追踪、个人生活计划、学习任务清单等

---

## 三、核心功能

### 3.1 TODO 任务管理（CRUD）

单个任务包含以下属性：

| 属性 | 类型 | 说明 |
|------|------|------|
| 标题 | 文本 | 必填，任务描述 |
| 完成状态 | 布尔 | 已完成 / 未完成 |
| 优先级 | 枚举 | 高 / 中 / 低 |
| 截止日期 | 日期 | 可选，任务截止时间 |
| 分类标签 | 标签数组 | 可选，支持多个标签 |
| 备注 | 文本 | 可选，任务详细说明 |
| 进度 | 数字 | 0-100，新建默认 0，完成自动 100% |
| 归档状态 | 布尔 | 已归档 / 未归档，默认未归档 |
| 排序序号 | 数字 | 手动拖拽排序用，新建自动递增 |

操作：创建、查看、编辑、删除任务；切换完成状态；拖动排序。

### 3.2 多列表管理

- 支持创建多个独立列表（如"工作"、"个人"、"学习"等）
- 每个列表包含各自的任务集合
- 支持列表的创建、重命名、删除
- 列表支持拖拽排序，持久化排序结果

### 3.3 任务进度

- 每个任务有 0-100 的进度值
- 新建任务默认 0%，标记完成自动设为 100%，取消完成重置为 0%
- 编辑任务时可拖动滑块调整进度
- 任务卡片显示绿色进度条

### 3.4 任务归档

- 编辑任务可勾选"已归档"
- 默认筛选视图仅显示未归档任务
- 筛选栏支持切换：未归档 / 已归档 / 全部
- 已归档任务不显示在周报中

### 3.5 拖拽排序

- 任务和列表均支持拖拽重排
- 排序规则：手动拖拽顺序 → 优先级（高→中→低）→ 更新时间倒序

### 3.6 周报生成

### 3.7 搜索与筛选

- **关键词搜索**：按任务标题/备注内容进行全文搜索
- **字段筛选**：按优先级、完成状态、归档状态、标签等字段组合筛选
- 默认筛选未归档任务
- 搜索与筛选可同时使用

### 3.8 设置页面

- 顶部齿轮图标进入设置
- 配置项：默认展开详情、默认优先级、默认筛选视图、删除前确认
- 数据管理：JSON 导入/导出、周报生成

### 3.9 数据导入/导出

- **导出**：将所有列表及任务数据导出为文件（JSON 格式），用于备份
- **导入**：从导出的文件恢复全部数据，支持合并或覆盖模式
- 入口位于设置页面

## 四、用户故事

| 编号 | 用户故事 | 验收标准 |
|------|---------|---------|
| US-01 | 作为用户，我能快速添加一条 TODO 任务 | 输入标题后回车或点击添加即可创建任务 |
| US-02 | 作为用户，我能为任务设置优先级、截止日期和标签 | 创建/编辑任务时可填写所有属性 |
| US-03 | 作为用户，我能标记任务为已完成或未完成 | 点击任务前的复选框即可切换状态 |
| US-04 | 作为用户，我能编辑或删除已有任务 | 每个任务提供编辑和删除操作入口 |
| US-05 | 作为用户，我能创建多个列表 | 提供新建列表功能，可命名列表 |
| US-06 | 作为用户，我能切换不同列表查看各自任务 | 列表切换后仅显示对应列表的任务 |
| US-07 | 作为用户，我能通过关键词搜索任务 | 输入关键词后实时过滤匹配的任务 |
| US-08 | 作为用户，我能按字段筛选任务 | 提供筛选条件选择，组合筛选结果 |
| US-09 | 作为用户，我能导出所有数据为文件 | 点击导出，下载 JSON 文件 |
| US-10 | 作为用户，我能从文件导入数据恢复 | 选择文件后导入，可选择合并或覆盖 |
| US-11 | 作为用户，我能为任务设置进度 | 编辑任务时可拖动滑块调整进度 0-100% |
| US-12 | 作为用户，我能归档任务 | 编辑时勾选归档，归档任务在周报中隐藏 |
| US-13 | 作为用户，我能拖拽排序任务和列表 | 拖拽手柄拖动，排序自动持久化 |
| US-14 | 作为用户，我能通过设置页面管理偏好 | 设置页配置默认优先级/筛选/展开/确认 |
| US-15 | 作为用户，我能生成并复制周报 | 点击生成周报，编辑后一键复制到剪贴板 |
| US-16 | 作为用户，我能按归档状态筛选任务 | 筛选栏支持未归档/已归档/全部切换 |

## 五、功能边界（明确不做）

- ❌ 云同步 / 多端同步
- ❌ 多用户协作 / 分享功能
- ❌ 推送通知 / 到期提醒
- ❌ 子任务 / 依赖关系
- ❌ 附件 / 图片上传
- ❌ Markdown 渲染（纯文本处理）

---

## 六、非功能性需求

- **离线运行**：所有功能在浏览器本地完成，无需网络
- **数据安全**：数据存储在浏览器本地存储中
- **性能**：任务列表操作响应时间 < 100ms
- **兼容性**：支持 Chrome、Edge 等主流 Chromium 内核浏览器
